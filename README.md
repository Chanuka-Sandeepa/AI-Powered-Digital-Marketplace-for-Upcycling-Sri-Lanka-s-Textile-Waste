# AI-Powered-Digital-Marketplace-for-Upcycling-Sri-Lanka-s-Textile-Waste
The system integrates computer vision, machine learning, and optimization techniques to create an intelligent textile waste management platform.

FINAL MEMBER DIVISION
Member 1 – Fabric Intelligence Module
Name: Sandeepa
Core Responsibility:
Fabric Inspection & Condition Intelligence System
Key Features:
Fabric Type Classification (Cotton, Polyester, etc.)
Color Detection
Quality Grade Prediction
Damage Detection (tear, stain, defects)
Fabric Health Score (0–100)
Repairability Prediction (repair / not worth)
Lifespan Estimation (remaining usable time)
Technologies:
CNN with Transfer Learning (ResNet / EfficientNet)
Optional: YOLO for defect detection
Description:
This module identifies fabric type and analyzes its condition by detecting defects, estimating quality, and predicting repairability and lifespan.

Member 2 – Sustainability Intelligence Module
Name: Siriwardhana
Core Responsibility:
Environmental Impact & Decision Intelligence System
Key Features:
CO2 Saving Prediction
Water Saving Prediction
Green Score
Scenario Simulation (reuse vs recycle vs discard)
Sustainability Score (0–100)
Best Action Recommendation
Trend Learning from historical data
Technologies:
Random Forest / XGBoost
Feature Engineering and Simulation Logic
Description:
This module predicts environmental impact and simulates different scenarios to recommend the most sustainable action.

Member 3 – Traceability, Recommendation & AI Pricing
Name: Madushanka
Core Responsibility:
Smart Value & Traceability Engine
Key Features:
AI Pricing Model
Recommendation System (buyers / reuse options)
Digital Material Passport
Transaction Traceability
Demand-aware pricing
Context-aware recommendations
Description:
This module calculates material value, provides recommendations, and maintains full traceability of textile waste transactions.

Member 4 – Marketplace & Optimization Intelligence
Name: Thalagala
Core Responsibility:
Marketplace Intelligence & Optimization System
Key Features:
Marketplace Platform (React UI)
Waste Forecasting
Smart Matching Engine (buyer-seller matching)
Distance Optimization
Cost Minimization
Environmental Benefit Maximization
Optimization using Weighted Scoring / Greedy Logic
Description:
This module manages the marketplace and uses optimization techniques to select the best buyer-seller matches efficiently.

Overall System Flow:
Fabric Image → CV Analysis → Sustainability Prediction → Optimization Decision → Marketplace Action
Summary:
The system integrates computer vision, machine learning, and optimization techniques to create an intelligent textile waste management platform.
